# Ahmed Ismail Portfolio

A static portfolio website ready for GitHub Pages.

## Deploy on GitHub Pages

1. Create a new GitHub repository.
2. Upload `index.html`, `style.css`, `script.js`, and the `assets` folder.
3. Go to **Settings → Pages**.
4. Source: **Deploy from a branch**.
5. Branch: **main** and folder: **/(root)**.
6. Save and wait for the GitHub Pages link.
